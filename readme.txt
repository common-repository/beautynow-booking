Plugin for searching and booking beauty appointments

== Description ==
Use the BeautyNow WordPress plugin to enable booking powered by BeautyNow on your WordPress site.

To learn more about BeautyNow, please visit our website at http://beautynowapp.com

If you have questions about using this widget, please contact support@beautynowapp.com

This code was derived from the `iframe` plugin built by `webvitaly`

== Screenshots ==
1. [booknow location_id="11664"] shortcode

== Installation ==
1. install and activate the plugin from your Plugins page
2. add shortcode `[booknow location_id="11664"]` to page or post content

== Notes ==
* `location_id` is mandatory and should be provided by support@beautynowapp.com
